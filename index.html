How to Setup
The Elderscrolls III: Morrowind Online

This is a guide on how to install, setup and play Morrowind Online.
1. Get a copy of Morrowind. [Option 1] [Option 2] . and install it, keep track of where you install it.
2. Install Chocolatey to quick install the next step, copy and paste into a command prompt open as administrator for quick easy installation. (Command Prompt as admin)
Use this command(double click the command and copy):
@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
3. Install OpenMW using the install script listed in the OpenMW chocolatey script(paste it in command prompt as admin).
Which should be: choco install openmw
4. Open OpenMW Wizard and search for where you installed your original copy of Morrowind.
5. Download TES3MP from the latest Windows release.
6. Place the files into a Morrowind multiplayer folder and you should have it all installed.
Now you should be able to play Morrowind online by clicking "openmw-launcher.exe". Make sure to adjust your options for your experience. After you have selected your options and hit play, you should have a server browser. Find the server you want to play on.
